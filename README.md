# clash-royale-meta-analysis-april-2026
Game analytics project using Python, SQL, web scraping, and visualization to analyze Clash Royale deck and card performance.

## Tools
- Python
- Pandas
- Numpy
- Plotly
- BeautifulSoup
- SQL

## Project Highlights
- Extracted and cleaned deck-level and card-level data
- Analyzed usage rate, win rate, rankings, and hero performance to support game balancing decisions
- Built SQL queries for game analytics use cases
- Created visualizations to communicate key insights
